Code Sample with Highlighting

This is a simple Python example:


def future_value(pv, r, n):
    """
    Compute future value of an investment.
    pv: present value
    r: interest rate per period
    n: number of periods
    """
    return pv * (1 + r)**n

print(future_value(1000, 0.08, 5))
Briefly explain what the function does and connect it to the financial formula shown on the next slide.

Mathematical Equation – Financial Formula

Future value of an investment:

$$ FV = PV \times (1 + r)^n $$

  • FV = Future Value
  • PV = Present Value
  • r = Interest rate per period
  • n = Number of periods
